# autoYYS
阴阳师副本自动化研究

## 前提条件
1. 开启Android模拟器，如BigNox
2. 启动阴阳师，将画面切换到探索场景（大地图）
3. 目标剧情以解锁（普通或困难）
4. 设置剧情菜单位置
5. 勾选锁定出战式神
6. 队伍实力可以稳定通关
7. 剩余体力充足

## 自动化基本思路

通过模拟手势操作，进入副本，攻击怪物，离开副本，再进入副本，以此循环

## 难点
1. 无法获取当前的场景和状态，只能通过有限的手势操作，实现副本自动化。
2. 当手势操作和当前场景不匹配时，可能进入并且卡在未知的状态

## 场景分析

场景宽度
地平线
剧情菜单
确认对话框
